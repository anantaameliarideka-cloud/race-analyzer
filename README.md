# 🏍️ MotoGP TRACK SIMULATOR

## 📖 Deskripsi

Aplikasi web untuk mensimulasikan lintasan balap MotoGP menggunakan konsep **Integral (Arc Length)** dari Kalkulus.

Aplikasi ini menghitung:
- ✅ Panjang lintasan yang berbentuk kurva
- ✅ Total jarak tempuh untuk sejumlah lap
- ✅ Estimasi waktu tempuh
- ✅ Tingkat kesulitan lintasan berdasarkan belokan

## 🎯 Fitur Utama

### 1. Pilih Bentuk Lintasan
5 jenis kurva untuk memilih:
- **Parabola** (y = ax²)
- **Sine Wave** (y = a·sin(bx))
- **Lingkaran** (x² + y² = r²)
- **Cubic** (y = ax³)
- **Exponential** (y = a·e^(bx))

### 2. Atur Parameter
- Sesuaikan parameter kurva
- Preview real-time visualisasi

### 3. Set Range Lintasan
- Tentukan dari mana sampai mana (dalam km)

### 4. Input Simulasi
- Jumlah lap
- Kecepatan rata-rata (km/h)

### 5. Hasil Simulasi
- 📏 Panjang lintasan per lap
- 📍 Total jarak untuk semua lap
- ⏱️ Estimasi waktu tempuh
- 📈 Tingkat kesulitan (Rendah/Sedang/Tinggi)
- 🎯 Visualisasi grafik lintasan
- 🔍 Analisis detail
